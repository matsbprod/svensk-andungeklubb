Adminhandledning — webbplatsens uppbyggnad och underhåll

Översikt

Webbplatsen är en statisk HTML/CSS/JS-sajt som versionshanteras i GitHub och automatiskt driftsätts via Netlify vid varje uppdatering av main-grenen.

Live-adress: andunge.se
GitHub-repo: github.com/matsbprod/andungeklubb
Netlify: app.netlify.com (team: 95:a)

Installera verktygen

Du behöver tre program för att kunna redigera och publicera sajten.

1. GitHub Desktop

Används för att hämta ner, hantera och ladda upp ändringar till repot.

Ladda ned GitHub Desktop →

  1. Installera och logga in med ditt GitHub-konto.
  2. Välj File → Clone repository och sök efter andungeklubb.
  3. Välj var du vill spara repot lokalt och klicka Clone.

2. Visual Studio Code

Textredigerare för att ändra i HTML/CSS/JS-filer.

Ladda ned Visual Studio Code →

  1. Installera och öppna programmet.
  2. Välj File → Open Folder och öppna mappen där du klonade repot.
  3. Alla filer syns i vänster filpanel.

3. GitHub-konto

Be Mats lägga till dig som collaborator på repot under Settings → Collaborators. Du får då en inbjudan via e-post som du accepterar.

Mappstruktur

  • index.html — startsidan
  • pages/ — alla undersidor (om-klubben, batregistret, butiken etc.)
  • pages/bat/ — individuella båtsidor (1.html–176.html)
  • css/ — style.css (global) och page.css (undersidor)
  • js/ — header.js, duck.js
  • img/ — bilder (hero, folk-grid)
  • img/boat/ — båtbilder (bat_001_2.jpg, bat_001_3.jpg...)
  • pdf/ — alla PDF-filer (regler, andungebladet, protokoll)

Publicera en ändring

Så här går det till när du gjort en ändring i en fil:

01Öppna filen i VS Code, gör ändringen och spara (Cmd+S).
02Öppna GitHub Desktop — du ser filen markerad som ändrad i vänster panel.
03Skriv en kort kommentar i fältet Summary, t.ex. "Uppdaterade båt 76".
04Klicka Commit to main.
05Klicka Push origin (uppe till höger). Netlify plockar upp ändringen automatiskt och sajten är uppdaterad inom ~30 sekunder.

Tips: Hämta alltid senaste versionen innan du börjar redigera — klicka Fetch origin i GitHub Desktop så du inte jobbar på en gammal version.

Vanliga uppgifter

Lägga till en ny ägare på en båtsida

Öppna pages/bat/[nummer].html i VS Code. Hitta tabellen under "Tidigare ägare" och kopiera en befintlig rad, ändra år och namn.

Uppdatera en annons på Köp och sälj

Öppna pages/kop-och-salj.html. Annonser ligger i div.info-box-block. Kopiera ett befintligt block och anpassa innehållet.

Lägga till en ny PDF i Andungebladet

Lägg PDF-filen i pdf/-mappen. Öppna pages/andungebladet.html och kopiera ett befintligt pdf-card-block, uppdatera filnamn, år och label.

Uppdatera lagersaldo

Gå till andunge.se/pages/lager.html och logga in med lösenordet. Saldot sparas automatiskt i browsern.
